Trusting yourself is not an innate ability; it is a skill that must be developed and refined over years.

    Growing in self-trust involves building competence in your desired area, and building competence looks like succeeding in the goals you set for yourself. So, how do you achieve this?

    Start out by choosing an area where you're failing and write out a measurable goal. Then, create a plan by breaking this goal down into smaller objectives. For example, if you want to exercise more, the plan could be as follows:

    • Exercise 1/week for a quarter (3 months).
    • Up the amount to 2/week for the next quarter.
    • Continue in this fashion until the desired n/week is attained.

    Here is another example: Let's say you want to improve your coding skills. A possible plan could be:

    • Find an interesting OSS project.
    • Use the software and chat with the community.
    • Find ways to contribute to the code base (documentation, PR's etc.).

    I spent a lot of time disappointed in myself for not completing goals because I bit off more than I could chew. Don't be like me: start small, work hard, and surprise yourself :)

    A second-order effect of building self-trust and confidence through competence is that you'll find yourself surrounded by individuals who are even more skilled than you. This exposure brings a new realization: no one should be idolized. At the end of the day, everyone is faking it a little bit, and you should be more forgiving with yourself.
